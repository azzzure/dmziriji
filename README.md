# useDES

使用 DES 来加密/解密。
密钥请放在当前目录下的 key.py 文件中。
明文放在 ./before 目录下。
密文放在 ./after 目录下。
运行：
```
加密 before -> after: 
python .\main.py YOUR_FILE_PATH 0
解密 after -> before: 
python .\main.py YOUR_FILE_PATH 1
